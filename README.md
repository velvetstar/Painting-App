<h1>Painting App</h1>

<h3>This is a blank canvas, where the user can draw/paint in different colors. By clicking on the background color-picker, the user can change the background color of the canvas. By clicking on the line color color-picker, the user can change the color of the paint. This app is designed so that the user has their pick of colors-every color under the sun-for both painting and the canvas background.</h3>
